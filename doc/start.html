Using GenChemLab

There are three menus when you start the program:

  • the file menu,
  • the experiment selection menu, and
  • the help menu.

    The file menu is always available. The file menu allows you to start a new experiment, close the current experiment, or close the program entirely.

    When the program is first started, select an experiment from the selection menu. This menu will be replaced by a menu specific to the experiment.

    The help menu is always available. The help menu opens general help, help for each experiment, and also displays program information.
